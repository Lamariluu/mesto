**Интро**

Сервис Mesto. Интерактивная страница, где можно редактировать профиль пользователя, добавлять фотографии, удалять их и ставить лайки. На странице применяется валидация форм.

**Технологии**
* HTML
* CSS
* JavaScript
* Flexbox-вёрстка
* Grid Layout
* Hover
* Opacity
* Использование методологии БЭМ
* Адаптивная вёрстка
